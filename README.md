# tw5-chart

A TiddlyWiki plugin for the [[Chart library|]].

Current library version: v4.2.1

Le plugin, dont voici [[le dépôt GitHub|https://github.com/nikorion/tw5-chart]], est une adaptation pour TiddlyWiki de la bibliothèque JavaScript [[Chart.js|https://www.chartjs.org/]] pour la création de graphiques. La version actuelle (0.1.0) du plugin s'appuie sur //Chart.js v.4.5.1// (octobre 2025). [[Lire sa documentation|https://www.chartjs.org/docs/4.5.1/]].

//Chart.js// est moins flexible et personnalisable que //D3.js// ou //EChart.js// et on n'y trouve que 8 types de graphes, mais son intérêt réside dans son accessibilité et sa légerté (200 ko, ce qui représente le 1/6e de ECharts qui allourdi TiddlyWiki de 150 % ! 😲). Il devrait satisfaire celui qui souhaite générer rapidement des graphes, moins l'artiste ou le bidouilleur. Pour eux, je ne peux que conseiller [[l'adaptation pour TiddlyWiki d'Echart.js|https://tiddly-gittly.github.io/tw-echarts/]] conçue initialement par Gk0Wk. Le plugin est aussi accessible depuis le gestionnaire CPL-Repo, avec l'avantage de proposer les mises à jour lorsqu'elles se présentent. L'adaptation officielle de D3.js, quand à elle, est en dormance.
